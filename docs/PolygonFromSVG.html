    PolygonFromSVG

    PolygonFromSVG

    Creates a new rigid body model with a regular polygon hull based on a SVG. The SVG parameter can either be an external SVG file, or an id of an embedded inline SVG element e.g. in index.html.

    Constructor

    # new PolygonFromSVG(world, attributes, optionsopt)

    Parameters:
    Name Type Attributes Description
    world Matter.World The Matter.js world object
    attributes object Visual properties e.g. position, radius and color
    options Matter.IChamferableBodyDefinition <optional>
    		 Defines the behaviour e.g. mass, bouncyness or whether it can move
    Source:
    • PolygonFromSVG.js, line 51
    Tutorials:
    • Tutorial: 5 - Complex path SVG Example Open preview , open code
    Examples 
    // SVG via the element id attribute, SVG is embedded in HTML.
const attributes = {
  x: 300,
  y: 500,
  fromId: "puzzle",
  scale: 0.6,
  color: "lime"
}

const options = {
  isStatic: true,
  friction: 0.0
}

let block = new PolygonFromSVG(world, attributes, options);
    // SVG via an external SVG file
const attributes = {
  x: 580,
  y: 710,
  fromFile: "./path.svg",
  scale: 0.6,
  color: "yellow"
}

const options = {
  isStatic: true,
  friction: 0.0
}

let block = new PolygonFromSVG(world, attributes, options)
